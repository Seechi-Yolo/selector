# 产品目标

## 问题与形态

在网页上标出元素、写说明，生成可粘贴给 AI 编程助手的提示词。**主交付**：浏览器扩展（建议 MV3）；选取与复制默认在本地完成。

## 能力边界

- **核心**：帮 AI 在代码里 **对上号**（selector、组件/文件线索等）。
- **样式**：计算样式节选、侧栏微调等，便于对齐视觉（见路线图第 4 阶段）。
- **交互**：**近期**不自动录制/回放真实数据流；**远期**支持用户 **手建交互流图** → 提示词（非自动扒页面）。

## 产品路线（与 [roadmap.md](./roadmap.md) 同步）

1. **✅ 阶段一** 扩展化（DDD + manifest，侧载可用）— 已完成  
2. 选取边界：页内相关 DOM 尽量全覆盖（iframe、Shadow 等，受权限约束）  
3. **设置与提示词模板**：默认拼接模板可改；**每条导出字段/板块** 可由用户在设置里开关或编辑  
4. 样式与高级提示词：复刻/参考样式、浏览器内手调、导出结构升级  
5. 设计系统：与 token/组件库对齐或从系统选样式  

**远期**（路线图外一行）：交互流图 → 提示词。

## 现状与交付形态

**主交付**为浏览器扩展（MV3）；安装与用法见仓库 `README.md`。若仍保留书签式注入，须与扩展 **同源构建**，权限与多 frame 行为以扩展为准。

## 非目标（节选）

近期不承诺自动还原完整交互与业务状态；不承诺商店时间与测试范围（团队另定）。
