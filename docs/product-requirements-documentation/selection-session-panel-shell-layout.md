# PRD：选取会话 — 壳体与布局（贴边条 + 悬浮已选）

**里程碑**：里程碑一（选取会话宿主 UI 形态）。事实口径：`README.md`、`manifest.json`。

**与交互/状态机**：键盘路径、说明层、剪贴板等行为见 [`selection-session-interaction-and-state.md`](./selection-session-interaction-and-state.md)。**产品故事入口**：[`user-stories-single-multi-and-layout-move.md`](./user-stories-single-multi-and-layout-move.md)。

---

## 1. 要做什么

- 选取会话的常驻宿主 UI **仅为贴页面边缘的窄条**（下称 **贴边条**）；**不存在**「展开为大体量矩形主面板」的形态，用户也**无需**在「收起 / 展开」之间切换来完成主路径。
- 当存在 **≥1** 项已选时，在贴边条 **朝向页面内容的一侧** 外缘外，呈现 **悬浮已选列表**（与条在布局上相邻；条在右沿时，列表在条的 **左侧**），用于标签/序号/焦点项等与选取集直接相关的信息。
- 贴边条内承载 **操作引导**（一主少辅，与交互 PRD **I-09** 一致）；**不再**使用「提示 / 当前选中」**Tab 切换**承载已选列表。

---

## 2. 区域与术语（壳体专用）

| 文档用语 | 界面用语（建议） | 说明 |
|----------|------------------|------|
| **贴边条** | **选取** 或产品名旁窄条语境 | 贴边常驻窄条；内含头部控件与操作引导区。 |
| **悬浮已选列表** | **当前选中**（列表标题可省） | **仅**在已选 **≥1** 时出现；**不在**贴边条内与引导抢纵向空间。 |
| **操作引导区** | **操作引导** | 位于贴边条内；等价于旧版「提示」Tab 区域，**无 Tab 标签**。 |

全局术语（修改说明、复制提示词、范围选取等）仍以交互 PRD 与故事文档为准。

---

## 3. 已定稿决策（壳体 L-xx）

| ID | 事项 | 决议 |
|----|------|------|
| L-01 | 形态 | **仅贴边条**；**无**「展开为大面板」态；主路径不依赖展开/收起切换。 |
| L-02 | 已选列表位置 | 已选 **≥1** 时显示 **悬浮已选列表**；置于贴边条 **靠页面内容侧** 外缘的相邻悬浮层（条在 **右** 沿时，列表在条 **左** 侧；若将来支持左沿条，则列表在条 **右** 侧）。列表与页上选取框 **分工**：列表管「条目与焦点」，页上管几何与 **Enter** 角标（见交互 PRD）。 |
| L-03 | 贴边条内容分区 | 条内 **上/边**：小尺寸头部 icon（**L-06**）；**主体**：**操作引导区**（**I-09**、**I-24**）。**无**「当前选中」Tab。 |
| L-04 | 贴边与磁吸 | 贴边条默认 **贴右**、**磁吸** `right: 6px`（数值可 CSS 微调）；垂直位置与避让不与画布选取框冲突为准。 |
| L-05 | 体量 | 贴边条为 **窄条**（相对历史「主面板」宽高显著减小）；以不遮挡页上选取与说明锚点为准，具体 CSS 由实现微调。**已选列表** 宽度与最大高度独立约束，超出可滚动，**不**把贴边条撑成「大面板」。 |
| L-06 | 头部 icon | **关闭**（结束会话或等价）为 **小尺寸** icon；**不再**提供「展开主面板」类 icon。若需「仅隐藏贴边条但会话仍存活」等能力，由路线图另文约定，**不**用旧版「最小化 ↔ 展开」语义。 |

---

## 4. 用户可感知行为

1. 进入选取会话后，用户始终只看到 **贴边条**（**L-01**、**L-04**、**L-05**）；无「从小条点成大板」的步骤。
2. 空选时：**不**展示悬浮已选列表（**L-02**）；引导文案见交互 PRD状态机。
3. 单选或多选 **≥1**：**悬浮已选列表**出现（**L-02**）；换焦点项时列表高亮与交互 PRD **当前焦点项** 一致。
4. 说明输入仍在 **画布选取框正下方**（交互 PRD **I-22**），**不在**贴边条或已选列表内。

## 5. 验收

- [ ] 无「展开为大矩形主面板」路径；贴边条为默认且唯一壳体形态（**L-01**）。
- [ ] 已选 **≥1** 时悬浮已选列表出现，几何关系符合 **L-02**；空选时不出现。
- [ ] 贴边条内为引导 + 小头部 icon（**L-03**、**L-06**）；无「提示/当前选中」Tab。
- [ ] 贴边 **L-04**、体量 **L-05** 与页上选取、说明层不恶性重叠。
- [ ] 画布下说明输入锚点仍符合交互 PRD **I-22**。
