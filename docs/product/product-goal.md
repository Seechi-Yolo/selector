# 产品目标

用户在网页上标出元素、撰写说明，并生成可粘贴至人工智能编程助手的提示词。主交付物为基于 Chrome Manifest V3 的浏览器扩展。元素选取与提示词复制在默认配置下于本地完成。

产品边界聚焦于为人工智能提供可定位的上下文，包括选择器、组件名称与源文件线索等。样式增强能力由路线图里程碑四承载。当前版本不实现真实操作流的自动录制。远期方向为用户手工编排交互步骤图并据此生成提示词，该能力独立于当前里程碑交付范围。

产品路线与 [roadmap.md](./roadmap.md) 所载里程碑保持一致：里程碑一为扩展化与领域模型拆分，已完成；里程碑二为选取范围；里程碑三为设置与提示词模板；里程碑四为样式与高级提示词；里程碑五为设计系统。

以下能力明确不在当前承诺范围内：自动还原完整业务交互；应用商店上架时间表与测试范围由团队另行管理。
