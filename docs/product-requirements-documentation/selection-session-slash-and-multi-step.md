## 1. 要做什么

- 提供 **`/`（Slash）** 或等价入口，在选取会话中唤起 **命令面板**（名称可落地为「命令」「快捷操作」等），用于缩短「选样式 → 选动作 → 指定参考」的多步路径。
- **选样式**：用户从设计系统（或团队约定清单）中选定一种 **组件/变体**（如按钮 primary / secondary），结果写入导出上下文与 **复制提示词**（与故事四一致）。
- **选动作**：用户选定一种 **动作模式**（如 **移动**）；在选定 **移动** 后，进入 **参考目标选取** 子步骤：用户在页上 **再点选**（或框选）**另一元素** 作为 **空间参考/目标**，与当前已选中的「被移动项」一并进入 **复制提示词**（与故事三一致）。
- Slash 面板与 **贴边条**、**悬浮已选列表** 在层级上 **不互相替代**：命令面板为 **短时覆盖层**；关闭后回到壳体 + 画布交互。

---

## 2. 用户可感知行为（草案）

1. 用户按 **`/`**（焦点在宿主可编辑区外、且会话规则允许时，细则与 **I-28** 不冲突为准）→ 出现命令面板。
2. 面板内可 **搜索/分组**：一类入口指向 **设计系统样式**（故事四）；一类指向 **动作**（故事三，首期可仅 **移动**）。
3. 选择 **样式** 后：面板关闭；所选样式令牌/名称进入会话状态；**复制提示词** 在下一次 **I-08** 刷新时包含该信息；用户仍可继续写 **修改说明**（**I-22**）。
4. 选择 **移动** 后：进入 **「请点选页面上的参考/目标元素」** 引导态；下一次有效点选 **追加或绑定** 为参考节点（与当前选取集的合并规则由实现定稿，须保证导出中 **被移动项** 与 **参考项** 均可定位）。
5. 用户可 **Esc** 取消子步骤或关闭命令面板，不清空已有选取（除非与全局 **I-11** 冲突，以不误清空为优先）。
