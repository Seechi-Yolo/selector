# 使用教程页 — 产品规划（文案与结构）

本文档描述「使用教程」页面的**信息架构与叙述顺序**，便于后续实现页面与替换 GIF。实现形态（扩展内 `options`/独立 HTML、或文档站）待定；**结构与文案以当前 Selector 行为为准**。

---

## 1. 教程要解决什么问题

用户第一次打开 Selector 时，需要快速知道：**从哪里开始、页面上哪些是扩展画的、每一块该怎么点、复制提示词怎么走通**。教程按「一条主路径 + 按界面拆块」组织，避免一次性堆满快捷键表。

---

## 2. 建议的阅读顺序（主路径）

1. **怎么打开**扩展（工具栏图标 / 快捷键）。
2. **右下角面板是什么**（总览：状态、说明、复制、列表）。
3. **在页面上怎么选元素**（单击、Shift、框选）。
4. **选中后页面上会出现什么**（实线框、标签、笔按钮）。
5. **怎么写每条元素的说明**（弹层、保存与清除）。
6. **怎么复制提示词**（按钮与快捷键、成功反馈）。
7. **常用补救操作**（暂停、撤销、清空、关闭扩展）。

每一块下面配 **一张 GIF 位**（先占位，后替换为录制素材）。

## 4. 分界面说明（教程正文草案）

以下为对用户展示的**简单中文**；若产品决定教程仅英文，可整段平移为英文，结构不变。

### 4.1 打开 Selector

- **是什么**：在任意网页上进入「选取模式」，扩展会在页面内注入界面。
- **怎么做**：点击浏览器工具栏里的 **Selector** 图标；或使用快捷键 **Alt+Shift+S**（以当前 manifest 为准）。
- **GIF**：`01-open-selector.gif`（展示从普通页到出现右下角面板与可选取状态）。

### 4.2 右下角面板（总览）

- **是什么**：固定在右下角的**小面板**，包含状态、操作提示、已选列表和「复制」按钮；**可拖动标题栏**移动位置。
- **怎么做**：先扫一眼面板，知道「状态在这里、复制在这里、已选元素列表在这里」。
- **GIF**：`02-panel-overview.gif`（整面板入镜，可轻微拖动一次）。

### 4.3 状态：Selecting / Paused

- **是什么**：标题旁的绿点 +「Selecting」表示正在选取；灰点 +「Paused」表示**暂停选取**（此时移动鼠标不会再改悬停目标，避免误触）。
- **怎么做**：需要暂时和页面正常交互时，按 **空格** 暂停；再按一次恢复。
- **GIF**：`03-status-pause.gif`。

### 4.4 面板标题栏：最小化、关闭

- **是什么**：一条可拖动的横条上的按钮；**最小化**只收起面板内容，**关闭**会退出本次选取（注入 UI 移除）。
- **怎么做**：最小化后仍可看到小条，需要时再展开；用完点关闭。
- **GIF**：`04-panel-chrome.gif`。

### 4.5 面板内的快捷键提示（只读）

- **是什么**：多行小字，概括当前版本支持的快捷方式（点击选、Shift 多选、方向键导航、空格暂停、复制、撤销、Esc 清空等）。
- **怎么做**：告诉用户「不用背表，用几次就熟；详细行为见下面几节」。
- **GIF**：`05-shortcuts-hint.gif`（可选：镜头扫过文字即可）。

### 4.6 悬停高亮（虚线框）

- **是什么**：鼠标在页面上移动时，扩展用**虚线框**标出当前指向的元素范围（排除扩展自身 UI）。
- **怎么做**：移动鼠标对准目标，确认框线是否对准你要的元素。
- **GIF**：`06-hover-highlight.gif`。

### 4.7 单击与 Shift 多选

- **是什么**：**单击**选中当前元素（会替换之前的单选）；**Shift+单击**在已有选区上**追加或切换**该元素（多选逻辑与当前实现一致）。
- **怎么做**：先单选精调，需要多个时再 Shift。
- **GIF**：`07-click-shift-select.gif`。

### 4.8 拖拽框选（矩形选区）

- **是什么**：在空白处按下左键拖拽，出现**半透明矩形**，松开后与矩形相交的有意义元素会进入选区（与 Shift 组合行为以当前实现为准，教程里写「框选一批」即可）。
- **GIF**：`08-marquee-select.gif`。

### 4.9 选中后的页面叠加层

- **是什么**：每个已选元素有**实线框**、**角点**、以及**小标签**（显示简短标识，如类名等）；便于确认选中了谁。
- **怎么做**：对照标签确认是否选对；不对则用面板列表移除或重新点选。
- **GIF**：`09-selection-overlay.gif`。

### 4.10 批注按钮与批注弹层

- **是什么**：选区旁的**笔形按钮**打开弹层；弹层里有输入框和 **Clear** / **Done**。
- **怎么做**：输入对该元素的修改说明 → **Done** 保存；**Clear** 清空该条说明；弹层打开时 **Esc** 可先关掉弹层（与清空选区的优先级在教程里写清：**先关弹层，再按 Esc 清空选区**——与当前 `selector-content-app` 行为一致）。
- **GIF**：`10-annotate-popover.gif`。

### 4.11 面板中的已选列表

- **是什么**：每条对应一个已选元素：序号、简短标签、有批注时带 **\***；右侧 **x** 移除单条；底部 **Clear** 清空全部。
- **怎么做**：用列表核对数量与顺序；与复制到剪贴板里的条目顺序一致感由文案点一句即可。
- **GIF**：`11-panel-tag-list.gif`。

### 4.12 复制提示词

- **是什么**：**Copy Prompt** 将当前选区与批注拼成结构化文本写入剪贴板；成功时按钮短暂显示 **Copied**（约 2 秒）。
- **怎么做**：选好、批注可选填 → 点按钮，或使用 **Cmd/Ctrl+C**（无 Shift，且有选中项时）。
- **GIF**：`12-copy-prompt.gif`。

### 4.13 方向键导航（单选时）

- **是什么**：仅当选中**一个**元素时，方向键在 DOM 树里上/下/左/右移动选中目标。
- **怎么做**：细调父、子、兄弟节点时用。
- **GIF**：`13-arrow-navigate.gif`（可选）。

### 4.14 撤销与清空

- **是什么**：**Cmd/Ctrl+Z** 撤销最近一次选区变化；**Esc** 在弹层打开时先关弹层，否则清空选区。
- **怎么做**：误操作时的退路。
- **GIF**：`14-esc-undo.gif`（可与 4.10、4.11 合并录制，减少条数）。

## 5. 教程页交互建议（产品层）

- **一页滚动**优于多步向导：用户可随时跳到「复制」「批注」相关章节。
- **每节结构统一**：标题 → 一句话「是什么」→ 2～4 步「怎么做」→ GIF → 「相关快捷键」一行（若有）。
- **首屏 CTA**：「30 秒走通：打开 → 点选一个元素 → 点 Copy Prompt」链到 4.1 + 4.7 + 4.12 的锚点。
- **与 README 关系**：README 保留表格化快捷键；教程页负责**带界面截图/动图的情境说明**，避免两处长篇重复，可在教程页脚链回 README。
